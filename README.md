Esta aplicación de finanzas personales, creada con PyQt5, nos permite registrar los gastos de forma sencilla. 

Podemos añadir una descripción, el monto exacto y la categoría a la que pertenece cada gasto
Ademas la aplicación valida que los datos que ingresas sean correctos y una vez que hayas registrado los gastos,
puedes verlos todos en una lista clara tambien ofrece un resumen detallado de tus gastos por categoría, mostrando el porcentaje que representa cada categoría del total gastado.
